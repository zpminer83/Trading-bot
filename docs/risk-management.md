# Risk Management

## Главная цель

Сохранение капитала важнее максимального объема торгов.

## Принципы

1. Бот не должен терять весь депозит.
2. Любое ограничение должно сначала снижать риск, а не останавливать торговлю.
3. Полная остановка — крайняя мера.
4. Размер позиции контролируется независимо от стратегии.
5. Один сбой API не должен приводить к остановке.
6. Все критические действия фиксируются в логах.
